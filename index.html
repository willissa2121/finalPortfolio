SandFit

A full stack fitness application for the everyday user. Sandfit has a wide array of pre built websties that users can select from based on muscle group and intenisty of workout. All workouts can be done without any equipment and in a small amount of space. Sandfit also tracks the users calorie goals for that day by using the Cunninghams equation to display their calorie goals, while also using the Edemam API to track the calories of food the user has eaten.

Role: Design, Server Management, Database

Carpoolify

A mobile first carpool app to help locals get discounted rides to common locations. The users may register as a driver or rider, if registering as a dirver, the users car year, make, and model are used to find the average price per mile cost of driving. The users can either start a ride or search for a ride. If a ride to a common location is available, users may populate an available ride until the number of seats is filled. Once the ride timer begins, google maps will route the travel destination, while also dropping pins on users that have requested a pickup

Role: Design, Web, User Authentication, Database
